# gtb2021

2000-2020 TB estimates


## Folders:

* ~/doc  
  documentation, data collection form

* ~/R  
  R scripts

* ~/data  
  latest data (binary)

* ~/csv  
  datasets with timestamp in file name (comma delimited text)

* ~/input  
  external input data

* ~/report  
  Global TB Report 2021 .rmd scripts to generate tables, figures and text
